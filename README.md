## What CDPaint Is For

CDPaint is a pixel art editor that keeps the familiar feel of MS Paint while focusing on the needs of retro-style artwork. It works in both browser and desktop setups, and it uses Tauri plus WebGL to deliver a responsive editing surface for sprites, palette-based images, and classic game graphics.

The app is a strong fit for Gen 3 ROM hacking, Game Boy Advance asset production, and retro game development in general. It combines straightforward drawing tools with more specialized editing functions, so you can move from rough concepts to game-ready assets without jumping between separate programs.

## How to Use It

Begin by creating or opening a canvas, then select the drawing mode that matches the task at hand. From there, place pixels, shapes, or selections as needed. When you are building assets for a game project, palette editing, sprite splitting, and image adjustments help shape the final output.

Typical workflow:
1. Open or create a canvas.
2. Switch between drawing and selection tools.
3. Edit the palette or reorder indices when working with indexed art.
4. Use tiled drawing mode for repeated patterns or structured layouts.
5. Export with the GBA or retro-focused tools when your asset is ready.
